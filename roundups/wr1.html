Week Roundup #1: June 17th - June 23rd

tl;dr Week Highlights

Monday 17th June

  • Worked on a process for testing and deploying custom seccomp profiles

    • Future work could be to automate testing of custom seccomp profile in a Kubernetes Pod via a CLI tool, requiring:

      1. Automatic discovery of a candidate Node in a cluster (available Pod space)
      2. Auto-labeling the candidate Node (e.g. seccomp-custom=true)
      3. Adding custom profile to the Node in /var/lib/kubelet/seccomp-profiles/<custom_profile_name>.json
      4. Generating Pod manifest with nodeSelector set to candidate Node’s label, and localhostProfile being set to profile filename
